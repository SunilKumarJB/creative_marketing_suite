AI-Powered Multi-Language Campaign Orchestration, Image Generation, and Style Cloning Architecture

Tech Stack: React, FastAPI, Vertex AI, Gemini 3.5 & 3.0

System Block Diagram

Layered Architecture Overview:

  • Client Tier (React): Composes inputs, custom logos, and reference creative visual files. Supports 15 aspect ratios and 4 resolution levels (up to 4K).
  • Application Tier (FastAPI): Routes user parameters and hosts assets helpers. Fallback servers mount compiled static assets directly on the root endpoint.
  • GCP Tier (Vertex AI): Integrates text orchestration (Gemini 3.5 Flash) and advanced image composition/editing (Gemini 3 Pro Image).

AI Campaign Builder (3-Step Pipeline)

Composing Ads from Scratch:

  • Step 1: Prompts Generation: Sends input to gemini-3.5-flash. Returns multi-language marketing taglines, base image prompt, and Nano-Banana edit/layout prompt.
  • Step 2: Base Image: Instructs gemini-3-pro-image to generate a clean, text-free, centered base product image matching resolution configurations.
  • Step 3: Overlay Editor: Uses gemini-3-pro-image to overlay the brand logo, render taglines, and embed the *T&Cs apply legal footer.

Multimodal Visual Remixer (Style Cloner)

Preserving Layout & Brand Elements:

  • Allows users to upload an ad creative as a visual layout reference.
  • Users can customize prompts (e.g. "move dish to corner", "change discount text", "apply Diwali elements").
  • FastAPI backend sends reference creative, brand logo, and prompts to gemini-3-pro-image.
  • Preserves spacing, font sizing, and visual boundaries while swapping assets dynamically.

Rate-Limit Resilience (Tenacity Retries)

Robust Transient Error Handling:

  • Catches API quota and concurrency errors (status code 429 or RESOURCE_EXHAUSTED).
  • Applies **Exponential Backoff** (starting at 2 seconds, backing off to a maximum of 12 seconds).
  • Performs up to 5 consecutive retry attempts.
  • Guarantees high workflow completion rates during high-traffic campaign seasons.
